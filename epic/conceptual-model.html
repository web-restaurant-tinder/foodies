Restaurant Tinder Conceptual Model

Profile

  • Profile Id (PK)
  • Profile Activation Token
  • Profile Email
  • Profile Hash
  • Profile Avatar
  • Profile Username
  • Profile First Name
  • Profile Last Name

Follow

  • Follow Profile Id (FK)
  • Follow Profile Id 2 (FK)

Restaurant

  • Restaurant Id (PK)
  • Restaurant Food Type
  • Restaurant Avatar
  • Restaurant Star Rating
  • Restaurant Name
  • Restaurant Address
  • Restaurant Latitude
  • Restaurant Longitude
  • Restaurant Phone
  • Restaurant Url

Swipe

  • Swipe Profile Id (FK)
  • SwipeRestaurantId (FK)
  • Swipe Yes or No

Relationship

  • A profile can follow many other profiles. (1/m)
  • A profile can have many swipes. (1/m)
  • A profile can swipe on many different restaurants. (1/m)
  • A swipe can happen on multiple restaurants. (1/m)
